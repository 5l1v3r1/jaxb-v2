Sources for the JAXB v2.0 Reference Implementation Project

This project contains the sources for v2.0 of the JAXB project at Java.Net.

This project is under the CDDL license.

Access to the JAXB v1.0 source code is available in the jaxb-sources project.

Build Instruction

You need to have Apache Ant 1.6 or later to build the JAXB RI. Go to the jaxb-ri directory and run ant dist to build the JAXB RI into the dist folder.

For convenient browsing of the source code, the project comes with NetBeans project files.
