Sources for the JAXB v2.0 Reference Implementation Project

Overview

This project contains the sources for v2.0 of the JAXB project at Java.Net.
  • Licenses: Dual license consisting of the CDDL v1.0 and GPL v2.
  • Governance: Same as Project GlassFish

Access to the JAXB v1.0 source code is available in the jaxb-sources project.

Build Instruction

You need to have Apache Ant 1.6 or later to build the JAXB RI. First check out the workspace: 

$ cvs -d:pserver:yourid@cvs.dev.java.net:/cvs co -d jaxb-ri jaxb2-sources/jaxb-ri

Go to the jaxb-ri directory and run ant dist to build the JAXB RI into the dist folder. 

$ cd jaxb-ri
$ ant dist

For those of you who hack the JAXB RI source code seriously, the project comes with NetBeans project files. (If you use other IDEs, please contribute the project setting to us!)
