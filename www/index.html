Access to the JAXB v1.0 source code is available in the jaxb-sources project under either the JRL or JDL licenses.

If you're viewing this project and have not agreed to the JRL please contact the project owner (listed above or send mail to owner@jaxb2-sources.dev.java.net) to have yourself removed from the project.

This project must be kept as a private project as that machinery is used to provide ACL-controlled access to the CVS repository.

Build Instruction

You need to have Apache Ant 1.6 or later to build the JAXB RI. Go to the jaxb-ri directory and run ant dist to build the JAXB RI into the dist folder.

For convenient browsing of the source code, the project comes with NetBeans project files.
